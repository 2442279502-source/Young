# 工程算量小助手 v2.0

Web 版建筑工程算量工具，支持**快速估算**和**精确配筋**双模式，覆盖板/梁/柱三种构件。计算符合 GB 50010-2010 / GB 55008-2021 等最新国标，精度可达施工备料级别。

## 功能

- **双模式**：快速估算（配筋率法）+ 精确配筋（按直径/间距逐根计算）
- **三种构件**：板（slab）、梁（beam）、柱（column）
- **精确计算**：混凝土方量、模板面积、钢筋重量与截面积、含钢量
- **明细输出**：每根钢筋的位置、直径、间距、根数、长度、重量
- **Excel 导出**：汇总表 + 钢筋明细表双 Sheet
- **成本预算**：支持混凝土/钢筋/模板单价输入，自动汇总费用

## 技术栈

| 层 | 技术 |
|----|------|
| 后端 | Python 3 + Flask |
| 前端 | HTML + CSS + Vanilla JS |
| 模板 | Jinja2 |
| 报表 | openpyxl |
| 标准 | GB 50010-2010 / GB 55008-2021 / GB/T 1499.2-2018 |

## 快速启动

```bash
# 1. 安装依赖
pip install flask openpyxl

# 2. 启动应用
python app.py

# 3. 浏览器打开 http://127.0.0.1:5001
```

```
└── static/

    └── app.js           # 交互逻辑
```
