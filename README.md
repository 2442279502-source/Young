# 🔍 智能图片目标检测 Web 应用

基于 **YOLOv8** 深度学习模型的图片目标检测系统，上传图片即可自动识别图中 80 种常见物体，并生成标注结果。

## 🎯 项目亮点

- 🧠 **AI 驱动**：使用 Ultralytics YOLOv8 预训练模型，无需自己训练
- 🌐 **Web 界面**：Flask + HTML/CSS/JS 全栈架构，浏览器直接操作
- 🎨 **美观 UI**：现代化渐变色设计，支持拖拽上传
- 📊 **直观结果**：标注框 + 中文标签 + 置信度百分比
- ⚡ **轻量快速**：YOLOv8n (nano) 版本，CPU 也能流畅运行

## 🛠️ 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 后端框架 | Flask 3.x | Python 轻量级 Web 框架 |
| 深度学习 | Ultralytics YOLOv8 | 目标检测模型，支持 80 类物体 |
| 图像处理 | OpenCV + Pillow | 图片读取、标注绘制 |
| 前端 | HTML5 + CSS3 + JavaScript | 响应式界面，拖拽上传 |
| 运行时 | PyTorch | 深度学习推理引擎 |

## 🚀 快速开始

### 1. 安装依赖
```bash
pip install flask pillow ultralytics
```

### 2. 启动应用
```bash
python app.py
```

### 3. 打开浏览器
访问 `http://127.0.0.1:5000`

### 4. 使用
- 上传一张图片（JPG/PNG/WEBP）
- 点击"开始检测"
- 查看标注结果和物体列表

## 📸 功能截图

启动后浏览器显示的主界面包含：
- **左侧面板**：图片上传区（支持拖拽）+ 原始图片预览
- **右侧面板**：标注结果图 + 检测物体列表（中文名 + 置信度）

## 📝 简历描述参考

> **项目名称**：基于 YOLOv8 的智能图片目标检测 Web 应用
>
> **项目描述**：独立开发了一个基于深度学习的图片目标检测系统，采用 Flask 框架搭建后端 API，集成 Ultralytics YOLOv8 预训练模型实现 80 类物体的自动检测与标注。前端使用原生 HTML/CSS/JS 实现了拖拽上传、实时预览、结果可视化等交互功能。支持 JPG/PNG 等多种图片格式，检测结果包含中文标签映射和置信度评分。
>
> **技术栈**：Python、Flask、PyTorch、YOLOv8、OpenCV、JavaScript、HTML5、CSS3

## 📄 项目结构

